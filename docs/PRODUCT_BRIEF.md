# 소견 — 소중한 견적 제품 기준서

## 서비스 개요

- **서비스명:** 소견 — 소중한 견적
- **제품 형태:** 모바일 우선 반응형 웹사이트 MVP
- **핵심 포지션:** 창업 조건을 정리해서 납품 상담을 제대로 받을 수 있게 도와주는 곳

소견은 아이스크림 창업을 준비하거나 기존 매장에 아이스크림 판매를 추가하려는 고객이 본인의 창업 조건을 단계별로 정리하고, 납품 상담에 필요한 기본 정보를 빠짐없이 준비할 수 있도록 돕는 웹사이트입니다.

## 1차 타깃

1. 체인점/전문점 체인 창업 희망자
2. 개인 아이스크림 전문점 창업 희망자
3. 기존 매장에 아이스크림을 추가하려는 점주

## MVP 목표

MVP의 목표는 **정확한 견적 금액 산출**이 아닙니다. 초기 버전은 고객이 납품 상담 전에 본인의 조건을 구조화해서 정리하도록 돕는 데 집중합니다.

- 창업 조건 정리
- 임시 보고서 생성
- 납품 상담 요청서 생성
- 이후 관리자 또는 파트너가 상담에 활용할 수 있는 기본 데이터 확보

## MVP 제외 범위

초기 MVP에서는 아래 기능을 구현하지 않습니다.

- 네이티브 앱
- 실시간 채팅
- 자동 업체 매칭
- 정확한 단가 비교
- 인테리어 업체 연결
- 부동산 중개
- 업체 로그인
- 결제 기능

## 커뮤니케이션 원칙

- 고객에게 확정 견적 또는 최저가 비교처럼 보이는 표현을 피합니다.
- 화면 문구는 “창업 조건을 정리해서 납품 상담을 제대로 받을 수 있게 도와주는 곳”이라는 핵심 포지션을 기준으로 작성합니다.
- 납품 상담 전 준비를 도와주는 서비스임을 명확히 전달합니다.
