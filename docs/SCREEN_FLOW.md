# 소견 화면 플로우

## 고객 플로우

1. 랜딩 페이지
2. 창업 종류 선택
3. 지역 선택
4. 매장 상태 선택
5. 매장 형태 선택
6. 키오스크/POS 선택
7. 소견 대표코드 안내
8. 납품 방식 선택
9. 쇼케이스/냉동고 지원 확인
10. 임시 보고서 생성
11. 상담 요청서 제출

## 단계별 선택지

### 1. 랜딩 페이지

- 소견의 핵심 포지션을 안내합니다.
- 핵심 문구: “창업 조건을 정리해서 납품 상담을 제대로 받을 수 있게 도와주는 곳”
- CTA 예시: “내 창업 조건 정리하기”

### 2. 창업 종류 선택

- 체인점/전문점 체인
- 개인 전문점
- 기존 매장에 아이스크림 추가

### 3. 지역 선택

- 시/도
- 시/군/구
- 동네/상권명 입력

### 4. 매장 상태 선택

1. 아직 매장을 찾는 중이에요
2. 보고 있는 매장이 있어요
3. 매장 계약을 완료했어요
4. 기존 매장에서 아이스크림을 추가하려고 해요
5. 기존 아이스크림 매장을 리뉴얼하려고 해요

### 5. 매장 형태 선택

#### 체인점/전문점 체인 선택 시

- 직원 상주형
- 테이크아웃 중심형
- 매장 취식 가능형
- 무인/반무인 체인형
- 아직 브랜드 상담 전이에요

#### 개인 전문점 선택 시

- 직원 상주형
- 테이크아웃 중심형
- 매장 취식 가능형
- 무인 운영형
- 반무인 운영형

#### 기존 매장에 아이스크림 추가 선택 시

- 냉동고만 추가
- 아이스크림 코너 추가
- 기존 POS에 같이 판매
- 별도 키오스크/POS 추가
- 배달/포장 메뉴로 추가

### 6. 키오스크/POS 선택

#### 1차 선택

- POS만 필요해요
- 키오스크가 필요해요
- POS + 키오스크 둘 다 필요해요
- 기존 장비를 사용할 예정이에요
- 아직 잘 모르겠어요

#### 많이 선택해요 옵션

- 카드 결제
- 간편결제
- 현금 결제 여부
- 영수증 출력
- 메뉴/상품 직접 수정
- 매출 확인
- AS 지원
- 월 사용료 확인

#### 현금 결제

- 현금 결제 필요 없음
- 키오스크에서 현금 결제까지 필요
- 현금은 직원/POS에서 처리
- 아직 모르겠어요

### 7. 소견 대표코드 안내

- 소견 대표코드로 상담받기
- 개별 상담으로 진행하기

### 8. 납품 방식 선택

- 여러 회사 제품을 같이 받고 싶어요
- 지역 대리점 상담을 받고 싶어요
- 아직 모르겠어요

### 9. 쇼케이스/냉동고 지원 확인

- 필요해요
- 이미 있어요
- 직접 구매할 예정이에요
- 아직 모르겠어요

### 10. 임시 보고서 생성

고객이 선택한 조건을 요약해서 상담 전 확인 가능한 임시 보고서를 생성합니다.

- 창업 종류
- 지역
- 매장 상태
- 매장 형태
- 키오스크/POS 필요 여부
- 납품 방식
- 쇼케이스/냉동고 지원 상태
- 상담 전 확인할 체크리스트

### 11. 상담 요청서 제출

임시 보고서 확인 후 고객이 상담 요청서를 제출합니다.

- 고객명
- 연락처
- 추가 메모
- 개인정보 및 상담 목적 안내 동의 문구
