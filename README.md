# 소견(Sogyeon)

소견 — 소중한 견적은 아이스크림 창업 조건을 정리해서 납품 상담을 제대로 받을 수 있게 도와주는 모바일 우선 반응형 웹사이트 MVP입니다.

초기 MVP는 정확한 견적 금액 산출보다 창업 조건 정리, 임시 보고서 생성, 납품 상담 요청서 생성을 목표로 합니다.

## 실행 방법

```bash
npm install
npm run dev
```

Windows PowerShell에서 `npm` 스크립트 실행이 막히면 아래 명령을 사용합니다.

```bash
npm.cmd install
npm.cmd run dev
```

개발 서버가 실행되면 브라우저에서 `http://localhost:3000`으로 접속합니다.

## 현재 구현 범위

- 랜딩 페이지
- 창업 조건 입력 단계형 플로우
- 창업 종류, 지역, 매장 상태, 매장 형태, 키오스크/POS, 소견 대표코드, 납품 방식, 쇼케이스/냉동고 지원 확인 화면
- 선택값 상태 관리와 임시 보고서 요약 화면

아직 Supabase, 로그인, 관리자 페이지, 실시간 채팅, 결제, 실제 상담 요청 제출 기능은 구현하지 않습니다.

## 개발 원칙

- 모바일 우선 반응형 웹사이트로 개발합니다.
- 기능 구현 전 `docs` 문서를 먼저 확인합니다.
- MVP 범위를 임의로 확장하지 않습니다.
- 화면 문구는 확정 견적이나 과장된 단가 비교처럼 보이지 않게 작성합니다.
