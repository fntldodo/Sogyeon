# 소견(Sogyeon)

소견 — 소중한 견적은 아이스크림 창업 조건을 정리해서 납품 상담을 제대로 받을 수 있게 도와주는 모바일 우선 반응형 웹사이트 MVP입니다.

초기 MVP는 정확한 견적 금액 산출보다 창업 조건 정리, 임시 보고서 생성, 납품 상담 요청서 생성을 목표로 합니다.

## 개발 원칙

- 모바일 우선 반응형 웹사이트로 개발합니다.
- 기능 구현 전 `docs` 문서를 먼저 확인합니다.
- MVP 범위를 임의로 확장하지 않습니다.
- 화면 문구는 확정 견적이나 과장된 단가 비교처럼 보이지 않게 작성합니다.
